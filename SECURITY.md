# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately:

* **Email**: Contact at [**this email**](mailto:anish.creations.hq@gmail.com?subject=YT%20Audio%20Air%20-%20Security%20Vulnerability%20Report)
* **GitHub**: Reach out via direct contact on GitHub or open a private security advisory.

Please do not open a public issue for security vulnerability reports.
